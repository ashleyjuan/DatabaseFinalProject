## 三、使用說明

### 可分為三方使用者：

### 管理者：

可查看訂單編號、包裹類型、金額、重量、目的地、抵達地、寄件人、收件人、寄件時間等資訊，另外可查看顧客過去一年包裹數量及消費金額、位在預計時間到達的包裹。
### 顧客：

* 查看訂單編號、包裹類型、重量、寄件時間、運輸狀態等資訊

* 進行訂貨、填寫包裹資料（開發中）

* 更改帳戶資料（開發中）


### 快遞：

查看送貨包裹資訊、更改包裹狀態。
